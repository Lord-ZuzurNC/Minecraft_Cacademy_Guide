# Minecraft_Cobblemon_Guide
A guide to install Cacademy as much similar as PokeRayou season 2

> Github limits the size of file to 100MB. Some files in the repo will be more than this limit so I splitted up these files into 100MB chunks.
> To join back these files into 1 you'll need to install [VSCode](https://code.visualstudio.com/) (but I personnally advise using [VSCodium](https://vscodium.com/)), install the "File & Folder Splitter Pro" plugin (it's free).
> You'll have to right click and "Join Split Files". I'll provide an exhaustive list of all splitted files bellow
> ![File and Folder Splitter Pro VSCode plugin](docs/Images/VSCode_File-Splitter.png)

## Eligos tutorial

### Quelques précisions avant de commencer

Lorsque le tuto indique un chemin de dossier, par exemple :
C:\Users\"user"\curseforge\minecraft\Instances\Cacademy - The Ultimate
Cobblemon Academy\config\cobbleride
"user" est à remplacer par le nom de votre session Windows.
Si vous vous appelez Timothée, cela donnera :
C:\Users\Timothée\curseforge\minecraft\Instances\Cacademy - The Ultimate
Cobblemon Academy\config\cobbleride
